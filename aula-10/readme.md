## Instalar Nodemon

```bash
npm i nodemon --save-dev
```

Editar no package.json

```json
"scripts": {
  "test": "echo \"Error: no test specified\" && exit 1",
  "dev": "nodemon app.js"
},
"nodemonConfig": {
  "ignore": [
    "node_modules"
  ]
},
```

Executar

```json
npm run dev
```
